# Shop-It: Mobile Self-Checkout

<https://shop-it-l9f0.onrender.com/> ↗️

---

## Problem

Having to wait in a long queue to pay for the long carefully picked items is boring and dissatisfying.

## Solution

Shop-It enables customers to scan item barcodes, view details, and make payments via Mpesa directly from their phones.

## Impact

- Improved customer satisfaction
- Reduced operational costs for counters or POS
- Faster shopping experiences
